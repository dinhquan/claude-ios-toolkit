# Update Code Maps

Refresh architecture and code maps with a focus on modern iOS data flow.

## Steps
1. Enumerate modules, targets, and entry points.
2. Document dependencies and data flow between modules.
3. Capture navigation structure and feature boundaries.
4. Update diagrams or markdown summaries.
5. Note current tech decisions and tradeoffs.

## Output
- A concise architecture overview.
- Updated module dependency map.
- Notes on data flow and ownership.
