# TDD Workflow (iOS)

1. Define behavior and acceptance criteria
2. Write failing XCTest(s)
3. Implement minimal code to pass
4. Refactor for readability and performance
5. Add edge cases and regression tests

Checklist:
- Deterministic tests
- Mocks for network and persistence
- No UI tests unless required
